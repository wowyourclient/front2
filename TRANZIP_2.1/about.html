About CodeFusion

CodeFusion is a team of dedicated individuals who work hard in creating web applications that are beneficial to the community. Codefusion was created as part of the Code Tennessee competition, in which we created Tranzip, an application designed to aid local students and parents in aquiring bus numbers and routes.
